## 15.1 nginx



### 15.1.1 nginx 缓存的bug

```shell
p：配置了 反向代理
location /api/ {
	proxy_pass http://127.0.0.1:8080/;
	proxy_set_header Host $host
}
但是请求图片的时候在 nginx就返回了404（不命中）

s：因为配置了 缓存
location ~ .*\.(jpg | gif)
{
	expires 30d
}
我们可以
location ^~ /api/ {
	proxy_pass http://127.0.0.1:8080/;
	proxy_set_header Host $host
}
#^~意思是在正则以前
#~意思是正则匹配
这样优先级就比较高了
```
