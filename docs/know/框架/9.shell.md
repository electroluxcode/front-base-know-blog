## 9.1 基础

### 9.1.1 循环

```shell
echo "请输入你想沟通的人数"
read n
count=1
while ((count<=n))
do  
    # do something
    
    sleep 2
    echo "当前沟通了: $count 人"
    ((count++))
    sleep 2
done
```

## 9.2  实用

### 9.2.1 查找当前文件夹 的 文件

```shell
ARRAY_NAME=()
ARRAY_NAME+=(*.sh)


count=1
echo ${ARRAY_NAME[0]} 

for i in "${ARRAY_NAME[@]}"; do 
    # > 是 强制覆写
    # >> 是 在尾部添加
    #  $(date +%Y%m%d%H%M%S) 对应着年 月 日 时分秒
    echo "$i $(date +%Y%m%d%H%M%S) $count " >> output.txt; 
    ((count++))
done
```
